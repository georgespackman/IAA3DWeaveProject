How to Install The Tool¶

  1. Ensure python packages listed in the documentation are installed using the Python Package Index (PyPI)

    eg. “pip install pymoo”

    Best practice is to install these using a virtual environment. https://docs.python.org/3/tutorial/venv.html

  2. Clone or download the repository from GitHub

    https://github.com/louisepb/IAA3DWeaveProject

  3. To use the tool follow the information in the online documentation, section: How To Use the Tool
